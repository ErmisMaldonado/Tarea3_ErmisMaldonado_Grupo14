# Tarea3_ErmisMaldonado_Grupo14
Este proyecto realiza el análisis de un conjunto de datos de comportamiento de clientes en comercio electrónico utilizando Apache Spark (PySpark).
Se ejecuta procesamiento en batch para identificar patrones de compra, comportamiento de clientes y abandono (churn).

# Análisis de Ecommerce con Apache Spark

## Dataset
Ecommerce Customer Behavior Dataset  
- Registros: ~50,000  
- Variables: 25  
- Formato: CSV  

## Tecnologías
- Python
- Apache Spark (PySpark)
- Pandas
- Matplotlib

## Ejecución
1. Clonar repositorio
2. Instalar dependencias
   pip install -r requirements.txt
3. Ejecutar el programa
4. python src/batch_analysis.py

## Funcionalidades del Proyecto
- Carga de datos desde archivo CSV
- Limpieza de datos (eliminación de valores nulos)
- Análisis exploratorio de datos (EDA)
- Agrupaciones y estadísticas (promedios, conteos)
- Visualización de resultados mediante gráficos

## Resultados Esperados
El sistema permite obtener:
- Número de clientes que abandonaron (churn)
- Promedio de compras por cliente
- Distribución de clientes por país
- Análisis del comportamiento de compra
